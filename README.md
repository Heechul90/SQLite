# SQLite3


## 1. 개요
## 1.1 경량 DBMS
- 별도의 서버가 필요 없음
- 모바일 기기에서 많이 활용되고 있음
- 파이썬3에 기본 내장되어 있음
- 파일 또는 메모리에 DB 생성
- [참고자료:](https://wikidocs.net/book/1530) SQLite로 가볍게 배우는 데이터베이스 (WikiDocs)

 ## 1.2 데이터 타입
- 동적 데이터 타입
- Null, Integer, Real, Text, Blob 유형이 있음 (Boolean, Date, Time 없음)
- 다른 유형 데이터를 삽입해도 컬럼에 맞게 알아서 들어감
- 다른 DB에서 사용하는 데이터유형 이름 그대로 사용해도 무방
